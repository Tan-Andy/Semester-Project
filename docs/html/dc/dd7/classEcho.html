The process in the echo class takes a vector containing floating points, then takes a given value of the vector multiplies it by a scalar gain, then adds it back into the vector after some delay. This creates an echo effect.

Parameters
inputAddress to a vector containing the sound data in floating point
gainFloating point scalar multiplier of how loud the echo should be, gain should be between 0 and 1 noninclusive
delayInteger value of how long after the initial sound should the sound be added back into the function, this value is in number of samples
numchannelsInteger value of the number of channels that sound is being sampled from, should be 1 for mono and 2 for stero this value is used to calculate the true delay of the sample in order to account for stereo files
Returns
std::vector<float>
